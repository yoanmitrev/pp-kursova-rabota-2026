   МАГАЗИН ЗА ТЕХНИКА – README
================================================

ОПИСАНИЕ
--------
Конзолно приложение за управление на малък магазин
за техника. Позволява добавяне на продукти, преглед
на наличността, извършване на продажби и справка за
реализираните приходи.

СТРУКТУРА НА ПРОЕКТА
--------------------
  tech_store/
  ├── models.py        – клас Magazin с всички методи
  ├── main.py          – главен файл с менюто
  ├── requirements.txt – използвани библиотеки
  └── readme.txt       – това ръководство

КАК СЕ СТАРТИРА
---------------
1. Уверете се, че имате Python 3.8 или по-нова версия:
       python --version

2. Отидете в папката на проекта:
       cd tech_store

3. Стартирайте програмата:
       python main.py

БИБЛИОТЕКИ
----------
Проектът използва САМО вградени модули на Python.
Не се изисква "pip install" на нищо.

ФУНКЦИОНАЛНОСТИ
---------------
1. Добавяне на продукт
   - Въвеждате: име, категория, цена, количество
   - Ако продуктът вече съществува, наличността се увеличава

2. Преглед на наличността
   - Сортиране по: Име / Цена / Количество

3. Продажба
   - Намалява наличността с продаденото количество
   - Проверява дали има достатъчно наличност

4. Справка за приходи
   - Показва общите реализирани приходи
   - Показва текущата стойност на склада

5. Търсене по категория
   - Филтрира продуктите по зададена категория
   - Примерни категории: Лаптопи, Телефони, Слушалки

КЛАС Magazin (models.py)
-------------------------
  __init__(ime)              – създава магазина
  dobavi_produkt(...)        – добавя/зарежда продукт
  pokaji_nalichnost(sort)    – извежда наличността
  prodaj(ime, kolichestvo)   – продава продукт
  spravka_prikhodi()         – справка за приходи
  tarsene_po_kategoriya(...) – търсене по категория
